# Projeto SQL Ateliê Lela

## Descrição
Banco de dados relacional para a loja de bolsas de crochê Ateliê Lela.

## Como executar
1. Abra SQLiteOnline: https://sqliteonline.com/
2. Execute o script `01_create_tables.sql` para criar as tabelas.
3. Execute o script `02_insert.sql` para inserir os dados.
4. Teste consultas com `03_selects.sql`.
5. Execute atualizações e exclusões com `04_updates_deletes.sql`.
